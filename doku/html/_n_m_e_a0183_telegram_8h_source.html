23char CheckSum(String NMEAData) {

37| | | | | \\
38* $--XDR,a,x.x,a,c--c, ..... *hh<CR><LF> \\
39
40 Field Number:
41 1) Transducer Type
42 2) Measurement Data
43 3) Units of measurement
44 4) Name of transducer
45 x) More of the same
46 n) Checksum

49 Temperatur $IIXDR,C,19.52,C,TempAir*19

51 Kraengung $IIXDR,A,0,x.x,ROLL*hh<CR><LF>
52*/
